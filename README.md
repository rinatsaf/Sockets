# Сетевой карточный блеф

Консольный сервер и клиент для простой сетевой карточной игры на двоих и более игроков. Сервер управляет раундами, раздаёт карты и рассылает состояние, а клиенты подключаются по TCP и отправляют команды через текстовый интерфейс.

## Требования
- .NET SDK 8.0 (см. `global.json`)

## Структура репозитория
- `Server/` — TCP-сервер, принимающий команды клиентов и управляющий логикой раундов.
- `Client/` — консольный клиент с интерактивными командами `play`, `accuse` и `leave`.
- `SemestrovkaSockets/` — общие модели игры (карты, игроки, ходы) для клиента и сервера.

## Быстрый старт
1. Установите .NET 8.0 и восстановите зависимости (при необходимости):
   ```bash
   dotnet restore
   ```
2. В одном терминале запустите сервер (порт 8080):
   ```bash
   dotnet run --project Server
   ```
3. В других терминалах запускайте клиентов и вводите имя при подключении:
   ```bash
   dotnet run --project Client
   ```
4. После подключения минимум двух игроков сервер начинает раунд и рассылает состояние игры.

## Игровой процесс
- В начале раунда создаётся колода из номиналов Ten–Ace и четырёх джокеров, перемешивается, каждому живому игроку выдаётся по 4 карты, а на стол автоматически выкладывается первая не-джокер карта.
- Ход игрока отправляется командой `play` в клиенте: выберите индексы карт, заявите номинал и количество — ход переходит следующему живому игроку.
- Любой игрок может бросить вызов предыдущему ходу командой `accuse`. Если заявленные карты не совпадают, обвиняемый рискует выбыть с вероятностью 1/6; при ошибочном обвинении такую же проверку проходит обвинитель. После проверки начинается новый раунд с пересдачей.
- Команда `leave` помечает игрока вышедшим и закрывает его соединение; при единственном оставшемся живом игроке сервер рассылает сообщение о победителе и завершает игру.

## Команды клиента
- `play` — сыграть выбранные карты и объявить номинал.
- `accuse` — обвинить предыдущего игрока во лжи.
- `leave` — выйти из партии и закрыть соединение.

Запускайте несколько клиентов, чтобы тестировать разные сценарии блефа и обвинений.
